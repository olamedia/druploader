Dr(ag and drop )uploader is a jQuery plugin to enable drag&drop upload for any element on the page.

With one line:

$(".dropzone").druploader();

Corresponding HTML code:

<div class="dropzone" drop-api="/upload-url">
CONTENT HERE, can include other drop zones
</div>

HTML attribute drop-api is an URL for file upload.

With optional html attributes:

<div class="dropzone" drop-api="/upload-url" 
drop-reload="{jQuery selector}" 
drop-reload-api="{URL where to get content to replace drop-reload element}">
CONTENT HERE, can include other drop zones
</div>

If 'drop-reload' attribute is found, then 'drop-reload-api' URL will be used to load contents for element found by 'drop-reload' jQuery selector.

Backend

  • Read filename from header, X-File-Name
  • Read file contents from POST body
  • If success, return json object with field status = 0. Else return different status.
